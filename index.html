            با GitHub Gist دیتات رو تو ابر ذخیره کن و بین گوشی/لپ‌تاپ
            sync کن.

            🔗 ساخت Token
             • 
            🔗 ساخت Gist
          

          

            
            
            

              از
              github.com/settings/tokens
              بگیر (فقط scope "gist" کافیه)
            

          

          

            
            
            

              یه Secret Gist بساز با محتوای {"init":true}، بعد ID
              رو از URL کپی کن

            1. نوع درخواست رو انتخاب کن

            2. اگه توضیح خاصی داری، بنویس

            3. کلیک "⚡ تولید متن"

            4. کلیک "📋 کپی"

            5. برو تو ChatGPT/Claude/Gemini، paste کن

            6. AI کاملاً می‌فهمه و کمکت می‌کنه!
